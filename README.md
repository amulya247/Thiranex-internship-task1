# Sales & Revenue Analysis Dashboard 📊

An interactive business intelligence dashboard built using Microsoft Power BI to analyze sales performance, revenue trends, and business insights.

## 🚀 Features

✨ KPI Tracking (Sales, Profit, Orders, Quantity)
✨ Revenue Trend Analysis
✨ Top Performing Products Visualization
✨ Region-wise Sales Analysis
✨ Interactive Filters & Slicers
✨ Data Cleaning using Power Query
✨ Business Insight Generation

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query Editor
* Excel / CSV Dataset
* Data Visualization

## 📈 Dashboard Insights

* Identified top revenue-generating products
* Analyzed sales growth trends over time
* Compared regional sales performance
* Created interactive reports for better decision-making

## 📂 Project Workflow

1. Imported raw sales dataset
2. Cleaned and transformed data
3. Created KPIs and charts
4. Added slicers and filters
5. Generated business insights
6. Designed an interactive dashboard
